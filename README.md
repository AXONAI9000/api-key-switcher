# AI9000 - AI工具集合

存放AI生成的各种实用工具。

## 目录结构

```
AI9000/
├── tools/          # 所有AI工具
├── templates/      # 工具模板
└── docs/           # 文档
```

## 工具列表

| 工具名 | 描述 | 语言 | 日期 |
|--------|------|------|------|
| [claude-key-switcher](tools/claude-key-switcher) | 可视化切换Claude API配置 | C# WPF | 2026-01-22 |

## 添加新工具

1. 在 `tools/` 下创建工具目录
2. 包含 README.md 说明用途和用法
3. 更新本文件的工具列表
